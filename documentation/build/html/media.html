This module contains useful objects for easy us of video and audio.

play(filepath=None, loop=False, volume=1, pitch=1, fadein=(0, 0), fadeout=(0, 0), callback=None)[source]¶

Method to play the an audio file.

Note

Current implementation doesn’t support fadein and fadeout overlap, make sure they don’t.

Parameters:
  • filepath (string) – Relative path (to the game folder) of the audio file to use.
  • loop (bool) – If true the audio will be played in loop.
  • volume (float) – The volume of the audio file relative to the master device.
  • pitch (float) – The pitch.
  • fadein (2-size-set) – The amount of time to wait until fadein and how long the fadein lasts (in seconds).
  • fadein – The amount of time to wait until fadeout and how long the fadeout lasts (in seconds).
  • callback (function) – Function to call once the playback ends.
